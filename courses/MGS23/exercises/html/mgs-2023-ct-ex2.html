Exercise 4: Interaction of products, coproducts, initial and terminal objects

Assume that the category C has a terminal object, for which we write 1, and an initial object, for which we write 0. Moreover, assume that C has all products and coproducts. This means that, for every pair of objects A,B ∈C0, we have a product diagram and a coproduct diagram. We write A × B for the “middle object” of the product diagram, and A + B for the “middle object” of the coproduct diagram.

Recall that we write XY if there is an isomorphism in C(X,Y ). For each of the following isomorphisms, either prove them in C or find a category C in which they don’t hold:

a. A × 1A

d. A + 0A

e. A × BB × A

f. A × (B + C)(A × B) + (A × C) Can you write down an isomorphism that never holds, i.e., that doesn’t hold in any category C?
